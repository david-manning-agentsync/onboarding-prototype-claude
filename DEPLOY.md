# Dev & Deploy Cheatsheet

## Test locally
npm run dev
→ opens at http://localhost:5173

## Deploy to GitHub Pages
git add .
git commit -m "describe your change"
git push
npm run deploy
→ live at https://david-manning-agentsync.github.io/onboarding-prototype-claude

## Notes
- `git push` saves code to GitHub
- `npm run deploy` builds and updates the live site
- Both are needed every time you want changes live
